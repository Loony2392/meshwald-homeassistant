# MeshWald

Home-Assistant-Integration für das [MeshWald](https://meshwald.de) LoRa-Mesh-Sensornetz
(Märkischer Kreis). Zeigt Temperatur, Luftfeuchte, Luftdruck und Luftgüte pro Node an.

Benötigt einen kostenlosen MeshWald-Account und einen persönlichen API-Key.

Nach der Installation:

[![Integration einrichten](https://my.home-assistant.io/badges/config_flow_start.svg)](https://my.home-assistant.io/redirect/config_flow_start/?domain=meshwald)
